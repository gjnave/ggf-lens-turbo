Lens Turbo U4 is a standalone, one-click desktop image-generation tool built around the
Lens-Turbo-SDNQ-uint4-static model by WaveCut:
https://huggingface.co/WaveCut/Lens-Turbo-SDNQ-uint4-static

Original model:
https://huggingface.co/microsoft/Lens-Turbo

The app provides prompt and negative-prompt entry, seed controls, random seed mode,
batch creation, and an integrated queue so new jobs can be added during generation.

The Queue tab shows running, pending, finished, and failed jobs. Pending jobs are saved
and restored after restart, and the last 10 finished jobs are remembered. Right-click
actions open or delete generated images and cancel queued work.

Generation settings include official Lens base-resolution presets and an experimental
custom-resolution mode with 32-pixel increments. Custom sizes are limited to
2560 x 1440, with a warning that sizes above the normal base-resolution range may
produce artifacts or cloned objects.

Fast resident-GPU mode is the default for repeat generation. Low-VRAM CPU offload is
available as a slower fallback because it releases pipeline modules between image
calls. The first generation still needs internet to download models and kernels unless
the portable cache was prepared in advance.

The Get Going Fast branded UI includes themes, tooltips, a preview panel, live logs,
an output-folder shortcut, used-seed display, and credit for DJ Grizzly:
https://www.youtube.com/@dj__grizzly
